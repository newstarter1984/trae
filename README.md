# 元宝学习小世界 🎮

一个有趣的汉字学习Web应用，带有"我的世界"风格主题！

## 功能特性

✅ **汉字学习** - 输入拼音或汉字，展示笔顺动画
✅ **大号字体** - 手机屏幕最大化显示汉字
✅ **拼音显示** - 展示汉字的拼音
✅ **组词示例** - 提供常用词语
✅ **10个主题** - 我的世界风格，自动切换
✅ **白天/黑夜** - 根据时间自动切换主题
✅ **PWA支持** - 可添加到主屏幕
✅ **20个Minecraft生物** - 在背景里走来走去！
